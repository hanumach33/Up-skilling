Exercise 24 - Serialize and Deserialize JSON Files

Objective

Learn how to serialize and deserialize objects using JSON in C#.

Scenario

Create a User object, serialize it to JSON, save it to a file, deserialize it back into an object, and display its properties.

Tasks

- Create a User class.
- Serialize the object to a JSON string.
- Save the JSON string to a file.
- Read the JSON file.
- Deserialize the JSON into a User object.
- Display the object's properties.

Files

- Program.cs
- user.json
