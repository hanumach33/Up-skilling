Exercise 23 - Simulate Async File Upload with Exception Handling

Objective

Learn how to work with asynchronous methods and exception handling in C#.

Scenario

Simulate a file upload operation by waiting for 3 seconds and return a success message. Handle any exceptions using try-catch.

Tasks

- Create an asynchronous method.
- Simulate file upload using `Task.Delay(3000)`.
- Return a success message after the delay.
- Handle exceptions using `try-catch`.
- Display the upload status.
