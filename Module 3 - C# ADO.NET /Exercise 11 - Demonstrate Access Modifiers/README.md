Exercise 11 - Demonstrate Access Modifiers

Objective

Understand the visibility and accessibility of public, private, and protected members.

Scenario

Create a base class with different access modifiers and demonstrate their accessibility from derived and non-derived classes.

Tasks

- Create a class with public, private, and protected members.
- Access the members from a derived class.
- Access the members from a non-derived class.
- Observe the visibility of each access modifier.
