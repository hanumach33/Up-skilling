Exercise 15 - Differentiate Abstract Classes and Interfaces

Objective

Understand the differences between abstract classes and interfaces.

Scenario

Create an abstract class and an interface, implement both in a derived class, and demonstrate polymorphism.

Tasks

- Create an abstract class `Vehicle` with an abstract method `Drive()`.
- Create an interface `IDrivable` with a method `Start()`.
- Implement both in a class `Car`.
- Demonstrate method overriding and polymorphism.
