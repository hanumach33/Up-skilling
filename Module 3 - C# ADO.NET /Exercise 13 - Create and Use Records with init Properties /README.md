Exercise 13 - Create and Use Records with init Properties

Objective

Understand the use of records and init properties in C#.

Scenario

Create an immutable Employee record using init properties and modify it using the with expression.

Tasks

- Create an Employee record with init properties.
- Instantiate the Employee record.
- Create a modified copy using the with expression.
- Display both objects to verify the original remains unchanged.
