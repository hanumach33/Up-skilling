Exercise 18 - Use the required Modifier in C# 12

Objective

Learn how to use the `required` modifier to enforce property initialization in C# 12.

Scenario

Create a Student class with required properties and initialize them while creating an object.

Tasks

- Create a Student class with required properties.
- Instantiate the Student class.
- Initialize all required properties.
- Observe the compiler behavior when a required property is not initialized.
