Exercise 12 - Use Auto-Properties and Backing Fields

Objective

Understand the use of auto-implemented properties and backing fields in C#.

Scenario

Create a Product class that uses auto-properties and a backing field with validation.

Tasks

- Create a Product class.
- Use auto-implemented properties for Name.
- Use a backing field for Price.
- Prevent negative values from being assigned to Price.
- Display the product details.
