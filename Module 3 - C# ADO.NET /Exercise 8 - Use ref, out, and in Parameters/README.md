Exercise 8 - Use ref, out, and in Parameters

Objective

Understand and demonstrate the use of `ref`, `out`, and `in` parameters in C#.

Scenario

Observe how each parameter modifier affects the values passed to methods.

Tasks

- Create a method using `ref`.
- Create a method using `out`.
- Create a method using `in`.
- Print values before and after each method call.
