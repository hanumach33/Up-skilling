Exercise 16 - Handle Null References Safely

Objective

Safely handle nullable references in C#.

Scenario

Create a Person class with nullable reference types and safely access its members using null-conditional and null-coalescing operators.

Tasks

- Create a Person class with nullable reference types.
- Use the null-conditional operator (`?.`).
- Use the null-coalescing operator (`??`).
- Demonstrate null checking for nullable references.
