Exercise 21 - Use Pattern Matching with is and switch

Objective

Learn how to use pattern matching with `is` and `switch` statements in C#.

Scenario

Create a method that accepts an object and determines its type using pattern matching.

Tasks

- Create a method that accepts an object.
- Use the `is` operator for pattern matching.
- Use the `switch` statement for pattern matching.
- Display appropriate information based on the object type.
