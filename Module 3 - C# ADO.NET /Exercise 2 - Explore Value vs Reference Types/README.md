Exercise 2 - Explore Value vs Reference Types

Objective

Understand the difference between value types and reference types in C#.

Scenario

Demonstrate how value types and reference types behave when passed to methods.

Tasks

- Create variables using value types (`int`, `double`).
- Create a custom class as a reference type.
- Create methods to modify both value and reference types.
- Display values before and after method calls.
