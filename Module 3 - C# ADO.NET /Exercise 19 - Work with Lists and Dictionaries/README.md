Exercise 19 - Work with Lists and Dictionaries

Objective

Understand how to work with `List<T>` and `Dictionary<TKey, TValue>` collections in C#.

Scenario

Create a List and a Dictionary, perform basic operations, and display their contents.

Tasks

- Create a `List<string>` and add values.
- Remove an item from the list.
- Display all list elements using `foreach`.
- Create a `Dictionary<int, string>`.
- Add key-value pairs.
- Remove an entry.
- Display all dictionary entries using `foreach`.
