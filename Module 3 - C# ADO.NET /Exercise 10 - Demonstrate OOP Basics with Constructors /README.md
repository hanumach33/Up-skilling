Exercise 10 - Demonstrate OOP Basics with Constructors

Objective

Understand object-oriented programming concepts using constructors.

Scenario

Create a Car class with both default and parameterized constructors.

Tasks

- Create a Car class with properties for Make, Model, and Year.
- Implement a default constructor.
- Implement a parameterized constructor.
- Create objects using both constructors.
- Display the details of both objects.
