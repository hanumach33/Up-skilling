Exercise 6 - Loop Through an Array with Different Loop Types

Objective

Practice iterating through an array using different loop constructs.

Scenario

Traverse an array using various loop types and apply conditions to skip or stop iteration.

Tasks

- Declare an integer array with at least 5 elements.
- Iterate using `for`, `foreach`, `while`, and `do-while` loops.
- Skip a specific value using `continue`.
- Stop the loop when a specific value is encountered using `break`.
