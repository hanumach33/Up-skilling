Exercise 14 - Demonstrate Inheritance and Method Overriding

Objective

Learn inheritance and method overriding in C#.

Scenario

Create a base class `Shape` and derive `Circle` and `Rectangle` classes that override the `Draw()` method.

Tasks

- Create a base class named `Shape`.
- Create derived classes `Circle` and `Rectangle`.
- Override the `Draw()` method in each derived class.
- Instantiate objects and call the `Draw()` method.
