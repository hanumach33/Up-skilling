Exercise 20 - Use LINQ for Filtering and Projection

Objective

Learn how to use LINQ to filter and project data from collections.

Scenario

Create a list of orders and use LINQ to filter orders based on the total amount and project selected properties into an anonymous type.

Tasks

- Create an `Order` class.
- Create a list of `Order` objects.
- Filter orders whose total amount is greater than 1000.
- Project the selected properties using an anonymous type.
- Display the filtered results.
