Exercise 22 - Create and Deconstruct Tuples

Objective

Learn how to create, return, and deconstruct tuples in C#.

Scenario

Create a method that returns multiple values using a tuple and deconstruct the returned tuple in the Main method.

Tasks

- Create a method that returns a tuple containing an integer and a string.
- Deconstruct the tuple in the Main method.
- Display the individual values.
