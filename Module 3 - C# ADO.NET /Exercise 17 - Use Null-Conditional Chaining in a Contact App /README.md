Exercise 17 - Use Null-Conditional Chaining in a Contact App

Objective

Implement null-conditional chaining to safely access object members and avoid null reference exceptions.

Scenario

Create a Contact class and safely display the contact's details using the null-conditional operator.

Tasks

- Create a Contact class with Name and PhoneNumber properties.
- Safely access the contact's name using the null-conditional operator (`?.`).
- Display the contact's details only if the object and its properties are not null.
- Use the null-coalescing operator (`??`) to display a default value.
