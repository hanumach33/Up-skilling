Exercise 7 - Implement Method Overloading

Objective

Demonstrate method overloading by creating multiple methods with the same name but different parameter lists.

Scenario

Calculate the total using overloaded methods with different parameter types and counts.

Tasks

- Create a method named `CalculateTotal(int, int)`.
- Create a method named `CalculateTotal(double, double, double)`.
- Call both methods from `Main()`.
- Display the calculated results.
