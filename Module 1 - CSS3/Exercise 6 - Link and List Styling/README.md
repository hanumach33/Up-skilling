Exercise 6 - Link and List Styling

Objective

Customize the appearance of links and lists using CSS.

Scenario

The default blue links and bullet lists do not match the portal design.

Tasks

- Style links using :link, :visited, :hover, and :active pseudo-classes.
- Customize list-style-type and list-style-position.
- Remove bullets from the navigation menu.
- Add padding and margin to list items.

Files

- index.html
- styles.css
