Exercise 1 - Why CSS

Objective

Understand the three different methods of applying CSS to a webpage.

Scenario

The designer wants you to experiment with different ways to apply styles.

Tasks

- Apply an inline style to make one heading red.
- Use an internal `<style>` tag in the `<head>` to define the body background.
- Link an external stylesheet named `styles.css`.
- Move all reusable styles to the external stylesheet.

Files

- index.html
- styles.css
