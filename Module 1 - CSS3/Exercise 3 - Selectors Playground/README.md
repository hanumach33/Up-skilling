Exercise 3 - Selectors Playground

Objective

Master different types of CSS selectors.

Scenario

You need to style various elements based on IDs, classes, and element types.

Tasks

- Use the universal selector (*) to reset margin and padding.
- Use an element selector to style all `<h2>` elements.
- Use an ID selector (`#mainHeader`) for the banner.
- Use a class selector (`.eventCard`) for event containers.
- Use a grouping selector (`h3, p`) to style multiple elements.
