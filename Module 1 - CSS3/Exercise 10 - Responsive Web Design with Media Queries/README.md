Exercise 10 - Responsive Web Design with Media Queries

Objective

Create a responsive webpage using CSS media queries.

Scenario

Users will access the Local Community Event Portal from mobile phones, tablets, and desktops.

Tasks

- Add a media query for screens smaller than 768px.
- Stack navigation links vertically on smaller screens.
- Reduce image and font sizes.
- Use %, vw, and vh units for flexible layouts.
- Use Flexbox for a responsive layout.
