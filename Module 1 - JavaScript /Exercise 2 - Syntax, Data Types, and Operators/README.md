Exercise 2 - Syntax, Data Types, and Operators

Objective

Use JavaScript data types, variables, operators, and template literals.

Scenario

Store event details like name, date, and available seats.

Tasks

- Use `const` for event name and date.
- Use `let` for available seats.
- Display event information using template literals.
- Use the increment (`++`) and decrement (`--`) operators to manage seat count.
