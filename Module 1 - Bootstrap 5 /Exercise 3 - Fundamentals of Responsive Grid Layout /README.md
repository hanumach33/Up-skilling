Exercise 3 - Fundamentals of Responsive Grid Layout

Objective

Learn how to build responsive layouts using Bootstrap's Grid System.

Scenario

Create a responsive layout that adapts to different screen sizes.

 Tasks

- Create a container with three columns.
- Stack columns vertically on mobile devices.
- Display two columns per row on tablets.
- Display three columns per row on desktops.
- Use `.container`, `.row`, and `.col-*` classes appropriately.
