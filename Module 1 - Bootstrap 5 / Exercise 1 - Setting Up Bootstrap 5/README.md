Exercise 1 - Setting Up Bootstrap 5

Objective

Learn how to set up Bootstrap 5 in a web page using the Bootstrap CDN.

Scenario

Create a basic HTML page and integrate Bootstrap 5 to use its components and utilities.

Tasks

- Create a basic HTML page.
- Link Bootstrap 5 CSS using the CDN.
- Link Bootstrap 5 JavaScript Bundle using the CDN.
- Create a sample Bootstrap button and alert.
