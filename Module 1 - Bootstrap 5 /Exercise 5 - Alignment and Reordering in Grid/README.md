Exercise 5 - Alignment and Reordering in Grid

Objective

Learn how to align and reorder Bootstrap grid columns.

Scenario

Design a responsive layout by centering content and changing the column order on different screen sizes.

Tasks

- Use `justify-content-center` to horizontally center the columns.
- Use `align-items-center` to vertically align the columns.
- Reorder columns using `order-md-2` and `order-md-1`.
