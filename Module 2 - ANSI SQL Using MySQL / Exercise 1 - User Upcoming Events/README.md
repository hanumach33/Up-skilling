Exercise 1 - User Upcoming Events

objective

Retrieve all upcoming events that a user has registered for in their own city.

Scenario

Display all upcoming events where the user's city matches the event city and sort the results by the event start date.

Tasks

- Display the user's name.
- Display the event title.
- Display the city.
- Display the event start date.
- Show only upcoming events.
- Sort the results by event start date.
