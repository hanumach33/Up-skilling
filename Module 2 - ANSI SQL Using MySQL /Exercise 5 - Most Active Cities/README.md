Exercise 5 - Most Active Cities

Objective

Identify the top 5 cities with the highest number of distinct user registrations.

Scenario

Generate a report showing the cities that have the most active event registrations.

Tasks

- Count the number of distinct users registered from each city.
- Display the city name.
- Display the total number of distinct user registrations.
- Show only the top 5 cities.
- Sort the results in descending order of registrations.
