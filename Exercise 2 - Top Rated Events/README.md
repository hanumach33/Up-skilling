Exercise 2 - Top Rated Events

Objective

Identify the highest-rated events based on user feedback.

Scenario

Display events with the highest average rating, considering only events that have received at least 10 feedback submissions.

Tasks

- Calculate the average rating for each event.
- Consider only events with at least 10 feedback records.
- Display the event title.
- Display the total feedback count.
- Display the average rating.
- Sort by average rating in descending order.
