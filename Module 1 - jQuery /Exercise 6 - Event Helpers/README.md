Exercise 6 - Event Helpers

Objective

Learn how to handle mouse and keyboard events using jQuery event helper methods.

Scenario

Create interactive elements that respond to mouse and keyboard actions.

Tasks

- Create a `<div>` with some text.
- Use `.click()`.
- Use `.dblclick()`.
- Use `.mouseenter()` and `.mouseleave()`.
- Use `.keypress()` on an input field.
- Make each event perform a visible action.
