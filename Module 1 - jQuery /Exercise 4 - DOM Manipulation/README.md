Exercise 4 - DOM Manipulation

Objective

Learn how to create and modify DOM elements using jQuery.

Scenario

Create a form that allows users to add items to a list and remove all items.

Tasks

- Create a form with an input field and a submit button.
- Read the value entered by the user.
- Create a new `<li>` element.
- Append the `<li>` to an unordered list.
- Add a "Remove All" button to clear the list.
