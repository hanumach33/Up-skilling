Exercise 1 - jQuery Library Inclusion

Objective

Learn how to include the jQuery library in an HTML page.

Scenario

Set up a webpage using jQuery and verify that it is working correctly.

Tasks

- Include jQuery using a CDN link.
- Include jQuery using a locally downloaded file.
- Create a paragraph displaying "Hello World!".
- Log the paragraph text to the browser console using jQuery.
