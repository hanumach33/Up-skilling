Exercise 2 - The $() Function

Objective

Learn how to select HTML elements using the jQuery `$()` function.

Scenario

Use jQuery to modify webpage content after the DOM has loaded.

Tasks

- Create a webpage with one `<h1>`, two `<p>` elements, and one `<button>`.
- Use `$(document).ready()` to ensure the DOM is loaded.
- Change the text of the `<h1>`.
- Hide one of the `<p>` elements when the button is clicked.
