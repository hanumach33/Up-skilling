Exercise 3 - jQuery Methods

Objective

Learn how to use common jQuery methods for showing, hiding, fading, and toggling elements.

Scenario

Apply different jQuery effects to multiple elements using button click events.

Tasks

- Create three `<div>` elements with the class `.box`.
- Use `.hide()`, `.show()`, `.fadeOut()`, `.fadeIn()`, and `.toggle()`.
- Apply each method using a separate button.
- Bonus: Chain `.slideUp().delay().slideDown()`.
