Exercise 5 - Working with Events

Objective

Learn how to handle user interactions using jQuery events.

Scenario

Change the background color of a div based on button click events.

Tasks

- Create a button with the id `colorBtn`.
- Create a div with the id `colorBox`.
- Change the background color of the div to red when the button is clicked.
- Change the background color back to white when the button is double-clicked.
